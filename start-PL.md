# Instrukcja instalacji i konfiguracji

1. **Pobierz repozytorium**  :
   ```bash
   git clone https://github.com/rafalstepkowski5-glitch/mod7-zad-1-find_friends_v4-binder.git
   ```

2. **Zainstaluj zależności** :
   ```bash
   cd mod7-zad-1-find_friends_v4-binder
   npm install
   ```

3. **Uruchom aplikację** :
   ```bash
   npm start
   ```

4. **Obsługa błędów** : W przypadku wystąpienia błędów, upewnij się, że masz zainstalowaną odpowiednią wersję Node.js oraz npm.
5. **Przeglądaj w przeglądarce**: Otwórz przeglądarkę i przejdź na `http://localhost:3000`.
